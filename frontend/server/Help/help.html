Claves Api

  • OllamaStudioUI puede funcionar sin claves API, pero para algunas funciones puede ser necesario introducir una clave API.
  • Las claves API pueden introducirce utilizando el boton de configuracion de la pantalla principal.
  • Clave API de Ollama (Gratuita). Necesaria para utilizar los modelos Cloud.
  • Clave API de Tavily (Gratuita). Necesaria para utilizar el motor de busqueda de Tavily. Muy recomendable.
  • Claves API de Google y Bing (Pago). Necesarias para utilizar los motores de busqueda de Bing y Google.
  • El motor de Duckduckgo funciona sin necesidad de clave API, recomendable solo para busquedas basicas.
